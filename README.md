## 🚀 Getting Started

### 1. Requirements
- Python 3.10+
- Node.js 18+

### 2. Setup
```bash
git clone https://github.com/the-shadow-0/DSA-Python.git
cd DSA-Python

# Install Backend
pip install -r api/requirements.txt

# Install Frontend
cd frontend && npm install
```

### 3. Launch
```bash
# Terminal 1: API
cd api && uvicorn main:app --port 8000

# Terminal 2: UI
cd frontend && npm run dev
```
